# Amazon Clone 🛒

A responsive Amazon homepage clone built using **HTML** and **CSS**. This project recreates the look and feel of Amazon's landing page to practice frontend web development and responsive design.

## 🚀 Features

- Responsive navigation bar
- Hero banner section
- Product category cards
- Multiple shopping sections
- Footer inspired by Amazon
- Clean and responsive UI

## 🛠️ Technologies Used

- HTML5
- CSS3

## 📂 Project Structure

```
amazon-clone/
│── index.html
│── style.css
│── images/
│── assets/
└── README.md
```

## 💻 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/amazon-clone.git
```

2. Open the project folder.

3. Run `index.html` in your browser.

## 🎯 Learning Objectives

- Practice HTML and CSS
- Improve responsive web design skills
- Understand layout design using Flexbox and CSS Grid
- Build a real-world frontend project
